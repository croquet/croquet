Example: Multiblaster

This version of Multiblaster specifies box: '/' in its Session.join() call instead of an API key to use this box as a reflector.

The box: '/' parameter is a shorthand notation for reflector: '/reflector', files: '/files' which allow you to specify reflector and file server URLs independently.
